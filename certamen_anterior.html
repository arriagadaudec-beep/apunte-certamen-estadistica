Pregunta 1

IC · Cociente de varianzas

Enunciado Se desea comparar el puntaje obtenido en una prueba estandarizada por estudiantes sometidos a dos métodos de estudio distintos. Se supone que los puntajes se distribuyen aproximadamente normal en ambos grupos. En cada grupo se obtuvo una muestra aleatoria de tamaño 7 y se calcularon los siguientes estadísticos:
EstadísticoMétodo AMétodo B
Media muestralx̄ = 78 ptsȳ = 68 pts
Desv. estándar muestralsA = 14 ptssB = 10 pts
Tamaño de muestranA = 7nB = 7

a) Compruebe el supuesto de igualdad de varianzas poblacionales mediante el intervalo de confianza para el cociente de varianzas $\sigma_B^2/\sigma_A^2$, al 95% de confianza.

Solución · Parte a IC al 95% para el cociente σ²B/σ²A

Identificarlos datos y el caso

Las varianzas poblacionales son desconocidas. Queremos ver si podemos asumir $\sigma_A^2 = \sigma_B^2$. Para eso construimos un IC para el cociente y verificamos si contiene el valor 1.

  • s²A14² = 196
  • s²B10² = 100
  • nA=nB7 → gl = n−1 = 6 para ambos
  • F crítico$F_{6,\,6;\,0{,}975} = \mathbf{5{,}82}$  (tabla F, 95% bilateral)

Fórmuladel IC para el cociente (muestras iguales)

Como $n_A = n_B$, el valor F crítico es el mismo en ambos extremos del intervalo:

$$IC_{95\%}\!\left(\frac{\sigma_B^2}{\sigma_A^2}\right) = \left(\frac{s_B^2}{s_A^2}\cdot\frac{1}{F_{6,6;\,0{,}975}}\;;\;\frac{s_B^2}{s_A^2}\cdot F_{6,6;\,0{,}975}\right)$$

Calcularel cociente muestral

$$\frac{s_B^2}{s_A^2} = \frac{100}{196} = 0{,}5102$$

Construirel intervalo

$$IC_{95\%} = \left(\frac{0{,}5102}{5{,}82}\;;\;0{,}5102\times 5{,}82\right)$$
$$IC_{95\%} = \left(0{,}0877\;;\;2{,}969\right)$$

Decisión¿contiene el 1?

Regla de decisión Si 1 ∈ IC → no se puede rechazar $\sigma_A^2 = \sigma_B^2$ → se asumen varianzas iguales.
Si 1 ∉ IC → se rechaza la igualdad → varianzas distintas (usar Welch).

Como $0{,}0877 < \mathbf{1} < 2{,}969$, el valor 1 pertenece al intervalo.

Conclusión IC = (0,088 ; 2,969) → 1 ∈ IC → Varianzas desconocidas pero iguales

Con un 95% de confianza, no hay evidencia suficiente para rechazar que $\sigma_A^2 = \sigma_B^2$. Se concluye que las varianzas son desconocidas pero iguales, lo que habilita el uso de la varianza pooled $s_p$ en los análisis siguientes.

Pregunta 2

PH · Diferencia de medias (2 poblaciones)

Enunciado Usando los mismos datos de la Pregunta 1 y la conclusión obtenida en ella, compruebe mediante un contraste de hipótesis la siguiente afirmación: "la diferencia de puntajes medios entre el Método A y el Método B es superior a 15 puntos". Use $\alpha = 0{,}05$.
Conexión con P1 De la Pregunta 1 sabemos que las varianzas son desconocidas pero iguales → se usa el estadístico t con varianza pooled $s_p$ y $gl = n_A + n_B - 2 = 12$.
Solución · Parte a Test t de 5 pasos · cola izquierda · δ₀ = 15

Paso 1Hipótesis

La afirmación es que la diferencia $\mu_A - \mu_B > 15$. Para verificarla, la ponemos como H₀ (lo que queremos "convencer de rechazar"). Si no se puede rechazar, los datos son consistentes con la afirmación.

$$H_0: \mu_A - \mu_B \geq 15 \qquad H_1: \mu_A - \mu_B < 15 \quad\text{(cola izquierda)}$$

Paso 2Nivel de significancia

α = 0,05. Prueba unilateral izquierda → valor crítico $-t_{12;\,0{,}95}$.

De tabla t: $t_{12;\,0{,}95} = 1{,}782$, así que el valor crítico es $\mathbf{-1{,}782}$.

Paso 3Estadístico calculado

Sub-paso A · Varianza pooled:

$$s_p^2 = \frac{(n_A-1)\,s_A^2 + (n_B-1)\,s_B^2}{n_A+n_B-2} = \frac{(6)(196)+(6)(100)}{12} = \frac{1{.}176+600}{12} = \frac{1{.}776}{12} = 148$$
$$s_p = \sqrt{148} = 12{,}166$$

Sub-paso B · Error estándar:

$$SE = s_p\sqrt{\frac{1}{n_A}+\frac{1}{n_B}} = 12{,}166\cdot\sqrt{\frac{1}{7}+\frac{1}{7}} = 12{,}166\cdot\sqrt{\frac{2}{7}} = 12{,}166\times 0{,}5345 = 6{,}503$$

Sub-paso C · Estadístico t (con δ₀ = 15):

$$t_c = \frac{(\bar x_A - \bar x_B) - \delta_0}{SE} = \frac{(78 - 68) - 15}{6{,}503} = \frac{10 - 15}{6{,}503} = \frac{-5}{6{,}503} = \mathbf{-0{,}769}$$

Paso 4Región de rechazo

Cola izquierda: rechazar H₀ si $t_c < -t_{12;\,0{,}95} = -1{,}782$.

Paso 5Conclusión

$$t_c = -0{,}769 \;>\; -1{,}782 \quad\Rightarrow\quad \textbf{No se rechaza }H_0$$
Decisión No rechazar H₀: μ_A − μ_B ≥ 15

Con α = 0,05, no hay evidencia estadística suficiente para afirmar que la diferencia de puntajes medios entre el Método A y el Método B sea superior a 15 puntos. Los datos son consistentes con la afirmación planteada.

Por qué H₀ es μ_A − μ_B ≥ 15 y no μ_A − μ_B = 15 El enunciado pide "comprobar" (verificar) que la diferencia es mayor a 15. Al plantear esto como H₀ (lo que asumimos cierto), una región de rechazo cola izquierda nos dice si los datos echan abajo esa afirmación. Como no la echan abajo, los datos son compatibles con la afirmación.
Pregunta 3

Muestreo · Tamaño de muestra (MAS · Proporciones)

Enunciado Una investigadora desea estimar la proporción de hogares de una ciudad que utilizan algún tipo de energía solar. El error de muestreo no debe exceder el 4% y se trabaja con un nivel de confianza del 95%. La investigadora no cuenta con información previa sobre la proporción real. La ciudad tiene aproximadamente 5.000 hogares. ¿Cuál debe ser el tamaño de muestra?
Solución · Tamaño de muestra MAS Proporciones · sin información previa · población finita

FiltroIdentificar el tipo de problema

El enunciado pide estimar una proporción → fórmula de proporciones. No hay subgrupos → MAS (no estratificado). La población es finita (N = 5.000) → se aplica el filtro del 5%.

Paso 1Cazar los datos

  • N5.000 hogares (población finita)
  • e4% = 0,04 (convertir a decimal; es proporción)
  • Zα/295% → 1,96
  • p · qSin información previa → p = q = 0,5 → p·q = 0,25
Por qué p = 0,5 "Sin información previa" es la señal para aplicar la ley del mínimo esfuerzo: se usa p = 0,5 porque entrega la variabilidad máxima posible (p·q = 0,25), garantizando que la muestra sea suficientemente grande para cualquier proporción real.

Paso 2Calcular el tamaño base n₀

$$n_0 = \frac{Z^2\cdot p\cdot q}{e^2} = \frac{(1{,}96)^2\times 0{,}25}{(0{,}04)^2} = \frac{3{,}8416\times 0{,}25}{0{,}0016} = \frac{0{,}9604}{0{,}0016} = 600{,}25$$

Paso 3Filtro del 5%

$$\frac{n_0}{N} = \frac{600{,}25}{5{.}000} = 0{,}120$$

Como $0{,}120 > 0{,}05$, la muestra base representa más del 5% de la población. Se aplica la corrección por población finita.

Paso 4Corrección y redondeo

$$n = \frac{n_0}{1+\dfrac{n_0}{N}} = \frac{600{,}25}{1+0{,}120} = \frac{600{,}25}{1{,}120} = 535{,}9$$
Tamaño de muestra (redondeo hacia arriba) n = 536 hogares

El tamaño de muestra necesario para estimar la proporción de hogares con energía solar, con error máximo del 4% y 95% de confianza, es de 536 hogares.

Efecto de la corrección Sin corrección (si la población fuera "infinita"), necesitaríamos 601 hogares. La corrección por ser una población finita de 5.000 lo reduce a 536, un ahorro de 65 encuestas (~11%). El motivo: cuando la muestra pesa más del 5% de la población, la población misma "ayuda" — no es necesario encuestar tanto porque la cobertura relativa ya es alta.
